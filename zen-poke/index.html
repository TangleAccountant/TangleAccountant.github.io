Indie Mobile Game

Zen Poke

A portrait-first needle-felting puzzle game in development for iOS and Android.

In Development iOS + Android (planned) Offline-first core loop

Zen Poke is built in Unity with a focus on calm pacing, readable progression, and touch-friendly mobile UX. The current production work is focused on shell polish, gameplay feel, and release-quality reliability.

What to Expect

  • Portrait-first mobile gameplay built for one-handed use
  • Needle-felting puzzle theme with a cozy visual style
  • Main, Daily, Shop, and Me shell tabs with progression-focused surfaces
  • Offline-first core gameplay loop
  • Optional ads for non-entitled users and a planned one-time remove-ads purchase

Monetization and Transparency

Zen Poke is currently in development and not yet released. The current release plan includes ad placements for non-entitled users and a one-time purchase that removes ads.

Before launch, this page and the Zen Poke privacy policy will be updated to reflect the final production SDK configuration and the exact ad/analytics providers enabled in release builds.
